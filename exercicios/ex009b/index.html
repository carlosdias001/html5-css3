Trabalhando com listas

Listas ordenadas

Para criar as listas usaremos a tag <ol> e para delimitar a lista a <li>

  1. Arcordar
  2. Tomar café
  3. Escovar os dentes
  4. Falcudade
  5. Almoçar

A tag <ol> possue parâmetro type , onde configuramos tipo de marcador da lista atual, que te da as opções de configuração numérica, alfabetica e números romanos.
Na lista em ordem numérica, podemos usar o parâmetro start para escolher o número de início.

Listas não oredenadas

Para criar listas não ordenadas usamos a tag <ul>

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Por padrão, a mesma usa o marcador de bolinha toda preenchida, mas podemos usar o parâmetro type tambem dentro da tag <ul>

Misturando as coias

Podemos também criar listas mistas, configurando listas dentro de listas.

  1. Item 1
  2. Intem 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
  5. Item 5
